# Bug Reports

This folder contains bug report templates and recorded issues found during testing.

How to report a bug
- Create a new file in this folder using the naming convention BUG-<id>-short-title.md
- Include clear steps to reproduce, expected vs actual results, environment, severity, and attachments (screenshots).

Severity levels
- Critical: Production-blocking or data loss
- High: Major functionality broken
- Medium: Partial loss of functionality or edge cases
- Low: Minor UI issues or typos
